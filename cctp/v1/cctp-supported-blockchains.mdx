CCTP V1 is available on the following blockchains where USDC is natively issued,
providing **Standard Transfer** functionality.

**Mainnet:**

- Aptos
- Arbitrum
- Avalanche
- Base
- Ethereum
- Noble
- OP Mainnet
- Polygon PoS
- Solana
- Sui
- Unichain

**Testnet:**

- Aptos Testnet
- Arbitrum Sepolia
- Avalanche Fuji
- Base Sepolia
- Ethereum Sepolia
- Noble Testnet
- OP Sepolia
- Polygon PoS Amoy
- Solana Devnet
- Sui Testnet
- Unichain Sepolia
